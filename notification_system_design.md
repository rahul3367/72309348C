# Notification System Frontend Design

## Overview

React, TypeScript, Vite, Material UI, Axios, and React Router are technologies used for frontend application development. The application offers a dynamic user interface that allows creating, viewing, filtering, and managing notifications. Logging middleware is incorporated into the application to log application activities and errors.

---

## Technology Stack

- React 18
- TypeScript
- Vite
- Material UI
- React Router DOM
- Axios
- Custom Logging Middleware
